# 📊 Miniguia de Estudos com NotebookLM: SQL e Análise de Dados no Mercado Financeiro

## 🎯 Contexto e Objetivos

Este projeto tem como objetivo explorar o uso da Inteligência Artificial como ferramenta de apoio aos estudos, utilizando o NotebookLM para organizar conteúdos, gerar insights e aprofundar conhecimentos.

O tema escolhido foi **SQL e Análise de Dados no Mercado Financeiro**, com foco em compreender como bancos utilizam consultas para análise de clientes, transações e risco.

### Objetivos de estudo:

* Aprender consultas SQL aplicadas ao contexto bancário;
* Entender análise de dados em operações financeiras;
* Utilizar IA para gerar resumos e explicações;
* Desenvolver pensamento crítico sobre respostas geradas por IA.

## 🤖 Engenharia de Prompts e Aprendizados ("Cicatrizes")

Durante o uso do NotebookLM, foram testados diferentes tipos de prompts para extrair melhores respostas.

### 🔹 Prompt 1:

**"Explique o que é SQL no contexto bancário"**

📌 Resultado:
Resposta muito genérica.

📌 Aprendizado:
Prompts muito amplos geram respostas superficiais.

---

### 🔹 Prompt 2:

**"Explique como SQL é usado para analisar compras de clientes em bancos, com exemplos"**

📌 Resultado:
Resposta mais detalhada e contextualizada.

📌 Aprendizado:
Adicionar contexto melhora significativamente a qualidade da resposta.

---

### 🔹 Prompt 3:

**"Crie exemplos de consultas SQL para identificar clientes com maior volume de gastos"**

📌 Resultado:
Resposta prática com exemplos úteis.

📌 Aprendizado:
Prompts práticos geram respostas mais aplicáveis.

---

### 🔹 Dificuldades encontradas:

* Respostas genéricas quando o prompt era pouco específico;
* Necessidade de validar informações;
* Ajustar perguntas para obter exemplos reais.

## 📘 Miniguia de Estudo

### 🧠 Resumo

SQL é uma linguagem utilizada para manipulação e consulta de dados em bancos de dados. No setor financeiro, ela é essencial para:

* Analisar transações de clientes;
* Identificar padrões de consumo;
* Avaliar risco de crédito;
* Gerar relatórios financeiros.

---

### 📖 Glossário

* **SQL (Structured Query Language):** Linguagem para consultar bancos de dados
* **Query:** Comando para buscar informações
* **Tabela:** Estrutura que armazena dados
* **JOIN:** Combinação de dados entre tabelas
* **WHERE:** Filtro de dados
* **GROUP BY:** Agrupamento de dados

---

### 🔁 Prompts Reutilizáveis

* "Explique [conceito] de forma simples com exemplo prático"
* "Mostre um exemplo de SQL aplicado a [situação bancária]"
* "Resuma este conteúdo em tópicos"
* "Explique como isso é usado no mercado financeiro"
